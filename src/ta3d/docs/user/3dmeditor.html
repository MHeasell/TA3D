3dMeditor

3DMeditor main window

It is the tool, designed specially for TA3D models operation. It allows shaders, texturing, adding transparency, scaling, converting models format, and periodically animation of 3D models (yeah, this allows to have animated map features, like spinning dishes of radar towers and animated trees).



3dmeditor included in every package of TA3D.
