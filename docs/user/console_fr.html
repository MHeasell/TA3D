Les commandes de la Console

Appuyez sur ")" sous Linux ou sur "`" ou "²" sous windows pour afficher/cacher la console pendant le jeu.
(n est un entier, f un nombre en virgule flottante)


1. fps_on/fps_off : afficher/cacher le nombre d'images par seconde

2. zshoot : capture du tampon de profondeur

3. video shoot : active/désactive l'enregistrement video

4. reload shaders : recharge les vertex et fragment programs utilisés pour le rendu de l'eau et des détails

5. show mission info : affiche les informations de mission et les drapeaux de l'unité sélectionnée

6. view debug : affiche sur la minicarte les zones dessinées par le moteur de rendu

7. ai debug : affiche des informations sur l'IA pour déboguer

8. internal name : affiche/cache le nom interne des unités

9. internal idx : affiche/cache le numéro de l'unité sélectionnée

10. exit : quitte le jeu

11. wireframe : active/désactive le mode fil de fer

12. priority n : 0, 1 ou 2, défini la priorité du programme (0:basse, 1:moyenne, 2:haute)

13. water quality n : 0, 1, 2, 3 ou 4 défini le niveau de détails du rendu de l'eau (0:faible, 1:normal, 2:haute, 3:très haute, 4:extrème, comme 3 mais avec tout les reflets)

14. shadow quality n : qualité des ombres (n=1: normal, plus n est grand, plus les ombres sont détaillées au détriment des performances)

15. shadow ray f : diffusion des ombres (nécessite une qualité d'ombrage suffisante)

16. shadow : active/désactive les ombres

17. details : (dés)active la texture de détails

18. particle : affiche/cache les particules

19. waves : affiche/cache les vagues sur les côtes

20. show script : affiche/cache les scripts liés aux unités sélectionnées

21. show model : affiche/cache les informations sur le modèle 3d de l'unité sélectionnée

22. rotate light : active/désactive la rotation de la lumière

23. shake : fait trembler l'écran comme lors d'une grosse détonation

24. freecam : active/désactive le mode caméra libre

25. fps_limit n : impose un fps maximum (n<=0 : aucune limite)

26. spawn nom_de_l_unité id_joueur nb_unités : crée les unité(s) spécifiées pour le joueur indiqué. Syntaxe:
nom_de_l_unité - spécifie le type d'unité par son nom, pratique pour tester ("any" = aléatoire)
id_joueur - spécifie le joueur qui controlera ces unités
nb_unités - nombre d'unités à créer

27. timefactor f : change la vitesse du jeu (1 = normal)

28. addhp n : ajoute n points de vie aux unités sélectionnées (n peut être négatif)

29. activate/deactivate : active ou désactive les unités sélectionnées

30. reset_script : arrête tous les scripts en cours d'éxécution pour les unités sélectionnées

31. unitinfo : affiche des informations sur les unités sélectionnées dans une console

32. kill : tue les unités sélectionnées

33. script n : lance le script numéro n sur chaque unité sélectionnée

34. pause : pause/reprendre la partie

35. give metal/energy/both player_id amount : donne les ressources amount au joueur player_id

36. metal : active/désactive le code triche du métal

37. energy : active/désactive le code triche de l'énergie
