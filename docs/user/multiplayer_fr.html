Multi-joueurs

Un clic sur le bouton Multijoueurs vous amène à l'écran réseau.

Si vous avez une connection à Internet vous verrez la liste de tous les serveurs Internet disponibles dans la partie gauche de l'écran.
Vous pouvez rejoindre n'importe quel serveur dans la liste, ou vous pouvez choisir de rejoindre un serveur spécifique en cliquant sur le bouton "rejoindre IP". Vous pouvez aussi créer votre propre serveur en cliquant sur "héberger".

Après création ou connection à un serveur vous arrivez dans l'écran de configuration de partie.
Il ressemble à celui du mode bataille, mais il y a 3 grandes différences:

- vous pouvez (dés)activer le bouton "serveur internet", il permet de rendre visible ou non votre serveur sur Internet.

- une interface de chat est disponible pour discuter avec les autres joueurs

- lorsque vous êtes près à jouer - cliquez sur le bouton à droite de votre pseudo.

Vous pouvez restreindre l'utilisation de certaines unités si vous êtes le serveur en cliquant sur le bouton "unités", ce changement affecte tous les joueurs, seules les unités disponibles pour tous seront utilisables pendant la partie.
Vous pouvez changer la limite d'unités et quelques autres paramètres.


Pour jouer en réseau il vous faut ouvrir le port 4242!
